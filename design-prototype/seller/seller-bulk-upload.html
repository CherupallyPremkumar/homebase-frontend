CSV Format Rules

Follow these guidelines to avoid upload errors

  • File format: UTF-8 encoded CSV with comma delimiters.
  • Required columns: Name, SKU, Price, Stock, Category.
  • SKU: Must be unique. Use format HB-XXXX-0000.
  • Price: Numeric only, no currency symbols. Minimum value 1.
  • Stock: Whole numbers only. Set 0 for out-of-stock.
  • Category: Must match an existing HomeBase category (Furniture, Lighting, Decor, Kitchen, Bathroom, Bedroom, Garden, Storage).
  • Max rows: 500 products per upload. Split larger files.
