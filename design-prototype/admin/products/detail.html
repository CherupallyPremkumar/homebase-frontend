Description

Bosch GSB 500W Impact Drill Kit is a versatile power tool designed for drilling into wood, metal, and masonry. Comes with a comprehensive 100-piece accessory set including drill bits, driver bits, and hole saws. Features variable speed control and forward/reverse functionality.

Key Features

  • 500W powerful motor for heavy-duty applications
  • Variable speed: 0-2600 RPM with lock-on switch
  • Impact drilling for masonry up to 13mm
  • 100-piece accessory kit included in carry case
  • Forward and reverse rotation for screwdriving
